# portfolio-ajay

SEO-friendly, fast, single-page portfolio (static HTML/CSS/JS) designed for GitHub Pages.

## Deploy on GitHub Pages

1. Push this repo to GitHub (e.g. `ajay9330/portfolio-ajay`).
2. In GitHub: **Settings -> Pages**
3. **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: `main` (or `master`) / root
4. Wait for the Pages URL to go live.

## Customize

- Main content: `index.html`
- Styling: `styles.css`
- Interactions (theme toggle + reveal): `script.js`
- SEO: `sitemap.xml`, `robots.txt`, meta tags in `index.html`
